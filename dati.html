Mappe e dataset

Mappa degli spazi verdi di Palermo

Gli spazi verdi pubblici sono stati mappati utilizzando due dataset complementari: OpenStreetMap (dati collaborativi aggiornati dalla community) e Copernicus Urban Atlas (dati satellitari ufficiali dell'Agenzia Spaziale Europea).

L'analisi riguarda esclusivamente il verde pubblico accessibile. Sono stati esclusi dall'analisi gli spazi privati e gli impianti sportivi.

Mappa della copertura arborea a Palermo

I dati di copertura arborea provengono dal Copernicus Tree Cover Density 2023, elaborato dall'Agenzia Spaziale Europea attraverso analisi di immagini satellitari ad alta risoluzione.

La mappa mostra il dataset raster originale con risoluzione di 10 metri. Ogni pixel rappresenta la percentuale di copertura arborea in quel punto, calcolata dalla densità delle chiome degli alberi visibili da satellite.

Legenda copertura arborea (% per pixel 10m)

<10% - Scarsa
10-20% - Ridotta
20-30% - Quasi sufficiente
30-40% - Sufficiente
40-50% - Buona
≥50% - Ottima

Dataset utilizzati

📍 Spazi verdi pubblici (OpenStreetMap)

Dataset degli spazi verdi pubblici nell'area di Palermo estratto da OpenStreetMap tramite Overpass API. Include solo spazi accessibili al pubblico, escludendo aree private o a pagamento.

Caratteristiche:
  • Nome dello spazio (quando disponibile)
  • Superficie in m²
  • Geometria poligonale per rappresentazione su mappa
File: osm_verde_palermo.geojson
Formato: GeoJSON

🛰️ Spazi verdi da satellite (Copernicus)

Dataset di spazi verdi derivato dall'analisi di immagini satellitari Copernicus Urban Atlas Land Cover/Land Use 2021. Identifica aree verdi attraverso classificazione del suolo, filtrato per includere solo spazi nel comune di Palermo.

Caratteristiche:
  • Nome dello spazio (quando disponibile)
  • Superficie in m²
  • Geometria poligonale per rappresentazione su mappa
File: copernicus_verde_palermo.geojson
Formato: GeoJSON

📊 Quartieri con dati aggregati

Geometrie dei quartieri amministrativi di Palermo arricchite con statistiche aggregate su copertura arborea, accesso al verde e popolazione. Utilizzato per visualizzazioni dashboard e mappe tematiche.

Caratteristiche:
  • Nome quartiere
  • Percentuale copertura arborea e classificazione
  • Popolazione residente (ISTAT 2021)
  • Percentuale popolazione coperta e distanza media da spazi verdi (dataset Copernicus)
File: quartieri_dati_completi.geojson
Formato: GeoJSON

🏘️ Sezioni di censimento (ISTAT 2021)

Geometrie delle sezioni di censimento ISTAT del comune di Palermo con dati demografici al 2021. Utilizzato per calcoli di copertura popolazione e analisi spaziali ad alta risoluzione.

Caratteristiche:
  • Codice sezione ISTAT
  • Popolazione residente per sezione
  • Geometria poligonale di dettaglio
File: sezioni_censimento_palermo.geojson
Formato: GeoJSON

📈 Statistiche quartieri (JSON)

File JSON strutturato con statistiche aggregate per quartiere, separato per dataset OSM e Copernicus. Utilizzato per popolamento dashboard e grafici interattivi senza necessità di elaborazione geometrica.

Caratteristiche:
  • Struttura: oggetto con chiavi "osm" e "copernicus"
  • Per ciascun quartiere: popolazione totale, popolazione coperta (entro 300m), percentuale copertura, numero spazi verdi
  • Popolazione identica in entrambi i dataset (fonte ISTAT 2021)
  • Formato compatto ottimizzato per caricamento web
File: quartieri_data.json
Formato: JSON

📝 Note metodologiche

Tutti i dataset sono disponibili nel repository GitHub del progetto e sono rilasciati con licenza open source. Le elaborazioni seguono gli standard WHO/EEA per la valutazione dell'accesso al verde urbano. La documentazione completa della metodologia è disponibile nel file METODOLOGIA.md.

Maps and Datasets

Map of green spaces in Palermo

Public green spaces have been mapped using two complementary datasets: OpenStreetMap (collaborative data updated by the community) e Copernicus Urban Atlas (official satellite data from the European Space Agency).

The analysis concerns exclusively accessible public green spaces. Private spaces and sports facilities have been excluded.

Green spaces - OpenStreetMap

Map of tree canopy coverage in Palermo

Tree canopy coverage data comes from Copernicus Tree Cover Density 2023, produced by the European Space Agency through high-resolution satellite imagery analysis. The map shows the original raster dataset with 10-meter resolution. Each pixel represents the percentage of tree canopy coverage at that point, calculated from the density of tree crowns visible from satellite.

Datasets used

All datasets are available in the project's GitHub repository and are released under open source licenses (CC BY 4.0 e ODbL 1.0). The processing follows the standards World Health Organization ed European Environment Agency for the assessment of urban green space access. Complete methodology documentation is available on the dedicated page.

📍 Public green spaces (OpenStreetMap)

Dataset of public green spaces in the Palermo area extracted from OpenStreetMap via Overpass API. Includes only publicly accessible spaces, excluding private or paid-entry areas.

Features:
  • Space name (when available)
  • Size classification
  • Area in m²
  • Polygonal geometry for map representation
File: osm_verde_palermo.geojson
Format: GeoJSON
License: ODbL 1.0

🛰️ Green spaces from satellite (Copernicus)

Dataset of green spaces derived from satellite imagery analysis Copernicus Urban Atlas Land Cover/Land Use 2021. Identifies green areas based on land classification, and is filtered to include only spaces in the municipality of Palermo.

Features:
  • Space name (when available)
  • Size classification
  • Area in m²
  • Polygonal geometry for map representation
File: copernicus_verde_palermo.geojson
Format: GeoJSON
License: CC BY 4.0

📊 Neighborhoods with aggregated data

Geometries of Palermo's administrative neighborhoods enriched with aggregated statistics on tree canopy coverage, green space access and population.

Features:
  • Neighborhood name
  • Tree canopy coverage percentage and classification
  • Resident population (ISTAT 2021)
  • Percentage of covered population and average distance to green spaces (Copernicus dataset)
File: quartieri_dati_completi.geojson
Format: GeoJSON
License: ODbL 1.0
View Download

📈 Aggregated statistics

Unified JSON file containing all project statistical data: general city data and aggregated data by neighborhood.

File structure:
  • City level (OSM/Copernicus): total spaces, % coverage, fortunate/disadvantaged population
  • Neighborhood level (OSM/Copernicus): population, covered population, % coverage, number of spaces
  • Neighborhood level (Tree Canopy Copernicus): % tree canopy coverage, classification, area km², deficit %, trees needed
File: quartieri_data.json
Format: JSON
License: CC BY 4.0
View Download
